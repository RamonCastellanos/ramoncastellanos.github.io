## 🚀 Proyectos destacados

### [Gapsi e-Commerce — Mantenimiento de Proveedores](https://github.com/RamonCastellanos/gapsi-ecommerce)
Sistema completo de mantenimiento de proveedores para tienda online.

- ✅ API REST con Spring Boot + H2
- ✅ Frontend React 19 con Redux y Material-UI
- ✅ Paginación, virtual scroll y validaciones
- ✅ PWA instalable en móvil
- ✅ Patrones de diseño: Strategy, Singleton, Observer

**Stack:** `React 19` `Spring Boot` `Redux Toolkit` `Material-UI` `H2` `MapStruct` `Lombok`
